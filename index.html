Bem-vindo!

Este site foi desenvolvido como parte do projeto da disciplina de Programação Front-End, do curso de Engenharia de Software da instituição UniCesumar.

O objetivo principal é aplicar os conhecimentos adquiridos sobre HTML5, CSS3, JavaScript e consumo de APIs em um site responsivo.

Funcionalidade principal: busca de endereço a partir do CEP utilizando a API pública ViaCEP.
